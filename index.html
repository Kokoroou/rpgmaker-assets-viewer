MakerLens
Browser-based viewer & extractor for RPG Maker game assets
Supported Engines
EngineEncrypted Extensions

RPG Maker XP / VX.rgssad  ·  .rgss2a
RPG Maker VX Ace.rgss3a
How to Use
  • Click Open Folder and select the game's root folder — the encryption key is auto-detected from System.json.
  • For RPG Maker XP / VX / VX Ace, use the Select archive button and pick the .rgssad / .rgss2a / .rgss3a file.
  • Browse folders in the sidebar, click any asset to preview it, then use the download button to save individual files.
  • Drag the sidebar edge to resize it. Resize and theme preferences are saved automatically.
🔒 100% Private & Offline: All decryption happens entirely in your browser — no files are ever uploaded, transmitted, or stored on any server. Your game data stays on your machine at all times.

⚖️ Legal: This tool is intended for personal use only, to access assets from games you own. Redistribution of extracted assets may violate the game's license or copyright.
